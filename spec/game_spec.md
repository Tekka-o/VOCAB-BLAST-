# VOCAB BLAST! - ゲーム仕様書（確定版）

本ドキュメントは、画面に表示された日本語に対応する英単語をタイピングして翻訳・入力する、英単語学習タイピングゲーム「VOCAB BLAST!」のゲームシステム、ルール、および内部仕様を定義した仕様書です。

---

## 1. ゲーム概要
プレイヤーの英単語翻訳力とタイピング速度・正確性を鍛えるゲームです。
画面中央に提示される日本語の意味に合う英単語を、制限時間内に正確にタイピングします。

---

## 2. 難易度（モード）仕様
プレイヤーはゲーム開始前に、以下の3つのモードからいずれかを選択してプレイします。

| モード名 | 出現する単語の条件 | モード説明 |
| :--- | :--- | :--- |
| **little** | 7文字以下の英単語のみ | 初級〜中級者向け。短い単語で素早く正確にタイピングするモード。 |
| **long** | 8文字以上の英単語のみ | 上級者向け。長い単語や複雑な綴りの単語に挑戦するモード。 |
| **remix** | すべての英単語（文字数制限なし） | 混合モード。短い単語から長い単語までがランダムに出現するモード。 |

---

## 3. ゲームルール・コアシステム

### 3.1 制限時間システム
* **初期制限時間**: **30秒**
* **時間延長仕様 (Time Bonus)**:
  * 単語を正解する（入力を完了する）たびに、その単語が属する分類（文字数）に応じて制限時間が以下のように増加します。
    * **7文字以下の単語 (little対象単語)**: **+1秒**
    * **8文字以上の単語 (long対象単語)**: **+2秒**
    * *※「remix」モード中であっても、出現した単語の文字数に応じて上記と同様に+1秒または+2秒が適用されます。*
  * **注意**: コンボ（連続正解）発生による残り時間の追加（時間延長）はありません。
* **ゲームオーバー**:
  * 制限時間が `0` 秒になった時点でゲームは終了（TIME UP）となり、リザルト画面へ遷移します。

### 3.2 スコアシステム
初期スコアは **0点** から開始し、以下の2つの方法でスコアが加算されます。

#### ① タイピング加点（1文字ごと）
* 正解のアルファベットキーを **1文字正しく打つたびに 100点** 加算されます。
* ※単語全体の完成を待たず、タイピングの進捗に応じて即座にリアルタイムでスコアが加算されます。

#### ② コンボボーナス（単語正解時）
* **コンボの定義**: 「1つの単語の最初の1文字目から最後の文字まで、一度も間違わずに（ミスタイプなしで）正解し、単語を完成させた場合」にコンボ数が `+1` されます。
* 一度でも間違ったキー（ミスタイプ）を押すと、その単語でのコンボ挑戦は無効となり、現在のコンボ数は `0`（リセット）になります。
* **加点タイミングと計算式**:
  * ミスせずに単語を完成させ、コンボ数が増加した瞬間に、以下のボーナススコアが即座に加算されます。
    $$\text{加算スコア} = \text{連続でミスせず正解した単語数（コンボ数）} \times 1000 \text{点}$$
    * *例：1問目ミスなし（1コンボ目） $\rightarrow$ +1,000点*
    * *例：2問目連続ミスなし（2コンボ目） $\rightarrow$ +2,000点*
    * *例：3問目連続ミスなし（3コンボ目） $\rightarrow$ +3,000点*
    * *※もし途中で1文字でもミスタイプをするとコンボ数は `0` に戻り、次の単語をミスなしで正解した際は再び 1コンボ目（+1,000点）からスタートとなります。*

### 3.3 タイピング入力ルール
* **ケースインセンシティブ (Case-Insensitive)**:
  * アルファベットの大文字・小文字は区別しません。大文字・小文字のどちらで入力しても正しく判定・受理されます。
* **ミスタイプのペナルティ**:
  * 間違ったキーを押した場合の「制限時間の減少」などのペナルティは一切ありません。
  * ただし、前述の「コンボ数」が即座に `0`（リセット）になります。

---

## 4. ランキング（データ保存）仕様
自分自身と向き合い、過去の自分の限界（ハイスコア）に挑戦することを目的としたランキングシステムです。

### 4.1 保存トリガーと保存先
* ゲームオーバー時、獲得した最終スコアが自動的にブラウザの **ローカルストレージ（localStorage）** に保存されます。

### 4.2 保存・表示データ制限
* 保存・管理されるデータは **スコア上位10件（TOP 10）** のみに厳選されます。
* 新しいプレイ結果のスコアがTOP 10に満たない（11位以下である）場合、そのデータは自動的に消去（破棄）されます。
* 記録・保存するデータ項目：
  1. 順位 (1位〜10位)
  2. プレイ時の獲得スコア
* **ライバルデータの非介入**:
  * 初期状態（初回起動時）はランキングデータは空（「No Record」などの表示）であり、他プレイヤーやNPC等のダミーデータは一切表示しません。完全にプレイヤー自身の過去の実績のみを記録・掲載します。
