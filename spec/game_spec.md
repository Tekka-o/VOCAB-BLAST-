# VOCABLAST! - ゲーム仕様書（確定版）

本ドキュメントは、画面に表示された日本語に対応する英単語をタイピングして翻訳・入力する、英単語学習タイピングゲーム「VOCABLAST!」のゲームシステム、ルール、および内部仕様を定義した仕様書です。

---

## 1. ゲーム概要
プレイヤーの英単語翻訳力とタイピング速度・正確性を鍛えるゲームです。
画面中央に提示される日本語の意味に合う英単語を、制限時間内に正確にタイピングします。

---

## 2. 難易度（モード）仕様
プレイヤーはゲーム開始前に、以下の3つのモードからいずれかを選択してプレイします。

| モード名 | 出現する単語の条件 | モード説明 |
| :--- | :--- | :--- |
| **little** | 7文字以下の英単語のみ | 初級〜中級者向け。短い単語で素早く正確にタイピングするモード。 |
| **long** | 8文字以上の英単語のみ | 上級者向け。長い単語や複雑な綴りの単語に挑戦するモード。 |
| **remix** | すべての英単語（文字数制限なし） | 混合モード。短い単語から長い単語までがランダムに出現するモード。 |

---

## 3. ゲームルール・コアシステム

### 3.1 制限時間システム
* **初期制限時間**: **30秒**
* **時間延長仕様 (Time Bonus)**:
  * 単語を正解する（入力を完了する）たびに、その単語が属する分類（文字数）に応じて制限時間が以下のように増加します。
    * **7文字以下の単語 (little対象単語)**: **+1秒**
    * **8文字以上の単語 (long対象単語)**: **+2秒**
    * *※「remix」モード中であっても、出現した単語の文字数に応じて上記と同様に+1秒または+2秒が適用されます。*
  * **注意**: コンボ（連続正解）発生による残り時間の追加（時間延長）はありません。
* **ゲームオーバー**:
  * 制限時間が `0` 秒になった時点でゲームは終了（TIME UP）となり、リザルト画面へ遷移します。
  * **ゲーム再開・画面遷移（リザルト画面からのナビゲーション）**:
    * プレイヤーがソフトロック（進行不能）にならないよう、リザルト画面には以下の2つのナビゲーションボタンが設置されています。
      * **PLAY AGAIN（もう一度遊ぶ）**: 現在選択している難易度モードのまま、即座に新しいゲームセッションを開始します。
      * **SELECT MODE（タイトル・モード選択へ戻る）**: ゲームのタイトル画面に戻り、難易度モードの変更やハイスコアランキングの確認が再び行えるようになります。

## 3.4 画面状態とUI連動仕様
ゲームの状態（ステート）に応じて、画面全体の要素がダイナミックに変化し、プレイヤーの集中力を高める設計となっています。

| ゲーム状態（ステート） | 表示されるコンポーネント | 非表示になるコンポーネント | 仕様説明 |
| :--- | :--- | :--- | :--- |
| **Title (タイトル・モード選択)** | ヘッダーロゴ、左サイドバー（モード選択、**振り返りモード開始ボタン**）、中央エリア（タイトルお題）、右サイドバー（TOP10ランキング） | ヘッダー内の「残り時間（Time）」および「スコア（Score）」 | ゲーム開始前のロビー状態。すべての記録、モード設定、および振り返りモードへのアクセスが可能です。 |
| **Playing (ゲームプレイ中)** | ヘッダーロゴ、ヘッダー内の「残り時間」「スコア」、中央エリア（タイピングお題、スペル入力、コンボ演出） | 左右 of サイドバー（モード選択、ハイスコア、TOP10ランキングすべて） | タイピングに100%集中できるよう、不要な情報をすべて非表示にした画面設計です。 |
| **Result (リザルト・ゲーム終了)** | ヘッダーロゴ、中央エリア（最終スコア、リトライ・タイトル戻るボタン） | ヘッダー内の「残り時間」「スコア」、左右のサイドバー（モード選択、ランキング等） | プレイ中の集中状態のレイアウト（サイドバーなし）を維持したまま、結果表示と次のアクションへの誘導を行います。 |
| **Review (振り返りモード)** | ヘッダーロゴ、中央・左右全体を広く使った**振り返り専用2カラムレイアウト画面**（ミスタイプ克服リスト、ヒント使用克服リスト、一括復習ボタン、戻るボタン） | ヘッダー内の「残り時間」「スコア」、通常の左右のサイドバーすべて | 苦手な英単語をリラックスして自分のペースでタイピング復習できる、専用の学習画面です。 |

### 3.2 スコアシステム
初期スコアは **0点** から開始し、以下の2つの方法でスコアが加算されます。

#### ① タイピング加点（1文字ごと）
* 正解のアルファベットキーを **1文字正しく打つたびに 100点** 加算されます。
* ※単語全体の完成を待たず、タイピングの進捗に応じて即座にリアルタイムでスコアが加算されます。

#### ② コンボボーナス（単語正解時）
* **コンボの定義**: 「1つの単語の最初の1文字目から最後の文字まで、一度も間違わずに（ミスタイプなしで）正解し、単語を完成させた場合」にコンボ数が `+1` されます。
* 一度でも間違ったキー（ミスタイプ）を押すと、その単語でのコンボ挑戦は無効となり、現在のコンボ数は `0`（リセット）になります。
* **加点タイミングと計算式**:
  * ミスせずに単語を完成させ、コンボ数が増加した瞬間に、以下のボーナススコアが即座に加算されます。
    $$\text{加算スコア} = \text{連続でミスせず正解した単語数（コンボ数）} \times 1000 \text{点}$$
    * *例：1問目ミスなし（1コンボ目） $\rightarrow$ +1,000点*
    * *例：2問目連続ミスなし（2コンボ目） $\rightarrow$ +2,000点*
    * *例：3問目連続ミスなし（3コンボ目） $\rightarrow$ +3,000点*
    * *※もし途中で1文字でもミスタイプをするとコンボ数は `0` に戻り、次の単語をミスなしで正解した際は再び 1コンボ目（+1,000点）からスタートとなります。*

### 3.3 タイピング入力ルール
* **ケースインセンシティブ (Case-Insensitive)**:
  * アルファベットの大文字・小文字は区別しません。大文字・小文字のどちらで入力しても正しく判定・受理されます。
* **ミスタイプのペナルティと視覚演出（確定仕様）**:
  * 間違ったキーを押した場合の「制限時間の減少」などのゲーム的なペナルティは一切ありません。
  * ただし、前述の「コンボ数」が即座に `0`（リセット）になります。
  * **視覚的フィードバック（演出）**:
    * プレイの集中力を削がないよう、**タイピング中にカード全体が激しく揺れたり動いたりするシェイク演出はいかなる場合も発生しません**。
    * 代わりに、ミスタイプが発生した瞬間、お題カードのボーダーが**一瞬赤くフラッシュ（ローズ色の境界線エフェクト）**し、背景もかすかに赤色を帯びます。
    * さらに、**現在入力待ちの文字カード（入力枠）自体も一瞬赤色の枠線とピンクの背景（ローズカラー）に変化**し、プレイヤーがどこを間違えたかをカードを揺らすことなく、直感的に100%識別できる設計となっています。
    * ※注意: **スペースキーによるヒント使用時は、この赤いフラッシュ演出は実行されず、プレイヤーが混乱しないよう穏やかに処理されます**。
* **苦手克服データの記録と自動克服仕様**:
  * 通常プレイ中、ミスタイプまたはヒント機能を使用した単語は、一時的な「振り返り（苦手）リスト」に自動登録されます（詳細は別途「振り返りモード仕様書（review_mode_spec.md）」に準拠）。
  * 登録されている苦手単語が通常プレイ中に再度出現し、プレイヤーがその単語を**一度もミスすることなく（ノーミスで）**タイピング完了できた場合、その単語は苦手リスト（ミスタイプ数・ヒント使用数とも）から**完全に自動削除**されます（完全克服機能）。

## 3.5 お助け・ヒント機能（確定仕様）
プレイヤーが英単語を思い出せない、あるいは綴りが分からず進行不能（詰み）になることを防ぐため、以下の2つの支援仕様を導入します。

### ① 英語スペルの1文字目（頭文字）の常時公開
* **仕様**: 出題された段階で、英単語の最初の1文字目（頭文字）がプレースホルダー上で常に公開されます。
* **表示・入力ルール（パターンB採用）**:
  * 1文字目はカード上に**薄いグレー（プレースホルダー）**で表示されます。
  * この1文字目は「自動入力」されているわけではなく、プレイヤーは通常通り**1文字目のキー入力から開始する**必要があります。

### ② スペースキーによるヒント表示機能
* **仕様**: プレイ中にスペルが分からなくなった際、キーボードの「**スペースキー**」を押すことで、ヒントを獲得できます。
* **表示・入力ルール（パターン2採用）**:
  * スペースキーを押すと、**次に打つべき1文字が、対応するカードに薄いグレーで表示（一時的なカンニング表示）**されます。
  * 実際の入力自体はプレイヤー自身がキーボードを打って行う必要があります。
* **ペナルティ・演出ルール**:
  * ヒント機能を使用した場合のペナルティとして、**「現在のコンボ数」が即座に `0`（リセット）**になります。
  * なお、「スコアの減点」や「制限時間の減少」などのペナルティはありません。
  * **演出制限**: ヒント使用時は、ミスタイプ時のような**赤い警告フラッシュ（赤枠、ピンク背景）は発生しません**（なお、ミスタイプ時であってもシェイク演出自体は集中力維持のため排除されています）。代わりに穏やかな「コッ」という効果音が鳴り、静かにヒントが表示されます。

## 4. ランキング（データ保存）仕様
自分自身と向き合い、過去の自分の限界（ハイスコア）に挑戦することを目的としたランキングシステムです。

### 4.1 保存トリガーと保存先
* ゲームオーバー時、獲得した最終スコアが自動的にブラウザの **ローカルストレージ（localStorage）** に保存されます。

### 4.2 保存・表示データ制限
* 保存・管理されるデータは **スコア上位10件（TOP 10）** のみに厳選されます。
* 新しいプレイ結果のスコアがTOP 10に満たない（11位以下である）場合、そのデータは自動的に消去（破棄）されます。
* 記録・保存するデータ項目：
  1. 順位 (1位〜10位)
  2. プレイ時の獲得スコア
* **ライバルデータの非介入**:
  * 初期状態（初回起動時）はランキングデータは空（「No Record」などの表示）であり、他プレイヤーやNPC等のダミーデータは一切表示しません。完全にプレイヤー自身の過去の実績のみを記録・掲載します。
