# VOCABLAST! - 振り返りモード（弱点克服）仕様書

本ドキュメントは、プレイヤーがタイピングミスをしたり、ヒント機能を使用したりした英単語をあとから効率的に復習・克服するための「振り返りモード（弱点克服モード）」に関する詳細なゲームシステム、データ管理、および画面遷移の仕様を定義したものです。

---

## 1. 振り返りモードの概要
ゲーム中に発生した苦手な英単語を自動で収集・分類し、プレイヤーが自分のペースでタイピング練習を行える専用のトレーニングスペースです。
タイマーやスコア、コンボといったプレッシャー要因を排除し、純粋にスペルを覚えることに集中できる設計となっています。

---

## 2. データの収集・記録仕様
苦手な単語のデータは、ゲームを起動している「そのセッション中（メモリ内）」にのみ保持されます（ブラウザのローカル保存 `localStorage` への書き込みは行いません）。

### 2.1 収集トリガー
プレイヤーが通常のゲームモード（little / long / remix）をプレイ中、以下のイベントが発生した際に対象の英単語（および日本語訳）が自動的に記録されます。

| 記録カテゴリー | トリガー条件 | 管理データ |
| :--- | :--- | :--- |
| **① ミスタイプ単語** | 対象の英単語の入力中に、一度でも間違ったキー（ミスタイプ）を押した場合 | **ミスタイプ数（回数）** を `+1` |
| **② ヒント使用単語** | 対象の英単語の入力中に、スペースキーによるヒント機能（一時的なカンニング表示）を使用した場合 | **ヒント使用数（回数）** を `+1` |

* ※1つの単語でミスタイプとヒント使用が同時に発生した場合は、双方のリストに登録され、それぞれのカウントが加算されます。
* ※同一セッション中に同じ単語で複数回ミス/ヒント使用があった場合、カウントは累積（加算）していきます。

---

## 3. 振り返り画面のレイアウト（左右2カラム分割仕様）
タイトル画面の「振り返りモード」ボタンから移行した専用の振り返り画面は、一目で苦手な単語を把握・分類できるよう、**左右2カラムの分割レイアウト**を採用しています。

```
+-----------------------------------------------------------------------+
|  [ ヘッダー: VOCABLAST! - 振り返りモード ]               [戻るボタン] |
+-----------------------------------------------------------------------+
|  [一括復習ボタン: 苦手単語をランダムに連続練習する]                   |
+-----------------------------------------------------------------------+
|  【左カラム: ミスタイプ克服リスト】  |  【右カラム: ヒント使用克服リスト】  |
|  * ミスタイプ数の多い順にソート      |  * ヒント使用数の多い順にソート      |
|                                      |                                      |
|  +--------------------------------+  |  +--------------------------------+  |
|  | [単語A] (日本語)  [ミス数: 5]  |  |  | [単語C] (日本語)  [ヒント数: 4]  |  |
|  | -> [練習ボタン]                |  |  | -> [練習ボタン]                |  |
|  +--------------------------------+  |  +--------------------------------+  |
|  | [単語B] (日本語)  [ミス数: 2]  |  |  | [単語D] (日本語)  [ヒント数: 1]  |  |
|  | -> [練習ボタン]                |  |  | -> [練習ボタン]                |  |
|  +--------------------------------+  |  +--------------------------------+  |
+-----------------------------------------------------------------------+
```

### 3.1 左カラム: ミスタイプ克服リスト
* **表示対象**: ミスタイプ数（回数）が `1` 以上の英単語。
* **ソート順**: **ミスタイプ回数が多い順（降順）**で並び替えられます。
* **表示情報**: 英単語、日本語訳、これまでのミスタイプ累計回数、および「個別練習ボタン」。

### 3.2 右カラム: ヒント使用克服リスト
* **表示対象**: ヒント使用数（回数）が `1` 以上の英単語。
* **ソート順**: **ヒント使用回数が多い順（降順）**で並び替えられます。
* **表示情報**: 英単語、日本語訳、これまでのヒント使用累計回数、および「個別練習ボタン」。

## 4. 復習タイピングシステム（練習中の特別ルール）
プレイヤーは、以下の2つの方法で苦手な単語をタイピング練習（復習）することができます。

### 4.1 練習モードの分類

#### ① 特定の単語を選んでタイピング（個別復習）
* 各単語のカードに配置された「個別練習（タイピング）」ボタンを押すと、その単語1件だけを対象としたタイピング入力が中央のメインカードに出現します。
* 1回タイピングを完了（正解）すると、個別練習は終了し、2カラムのリスト画面に戻ります。

#### ② リスト全体の単語がランダムな順番で出てくる機能（一括復習）
* 画面上部に配置された「一括復習スタート」ボタンを押すと、リスト内の全単語が自動的にシャッフルされ、ランダムな順番で1問ずつ連続して出題されます。
* 登録されている全単語を一通り1回ずつタイピング完了すると、一括復習は終了し、リザルトを挟まずに2カラムのリスト画面に戻ります。

### 4.2 練習中の特別ルール
復習タイピング中（個別復習・一括復習のいずれも）は、プレイヤーがリラックスしてスペルの確認を行えるよう、以下の特別ルールが適用されます。

1. **時間制限・スコア・コンボの排除**:
   * 画面上に制限時間（タイマー）やスコア、コンボ数の表示は一切存在せず、時間切れによるゲームオーバーもありません。
2. **ヒント機能の利用可能**:
   * 通常ゲーム時と同様に、最初の1文字目のプレースホルダー表示、およびスペースキーによるヒント表示（次に打つべき1文字を薄グレーで表示）が利用可能です。
3. **ミスタイプのカウント除外**:
   * **練習中にタイピングを間違えても、苦手データの「ミスタイプ数」は加算（カウントアップ）されません**。これにより、苦手克服のための練習中にさらに苦手度が増すというネガティブフィードバックループを回避します。

---

## 5. 苦手履歴の保持仕様（克服判定 of 非適用）
本ゲームでは、練習や本番でタイピングを成功させても、苦手リストの各回数が減少したり、自動的にリストから削除されたりすることはありません。

### 5.1 履歴の累積保持
* 練習を繰り返したり、本番ゲーム中にノーミスでタイピング成功したとしても、一度記録されたミスタイプ回数やヒント使用回数は減少せず、リストから自動的に削除されることもありません。
* これにより、プレイヤーは現在のセッション中（起動中）に、どの単語を何回間違えたか、どの単語で何回ヒントを使用したかの正確な累積履歴をいつでも「振り返りモード（弱点克服ルーム）」で把握・確認することができます。
* 苦手リストは、ブラウザの再読み込みやタブを閉じることでセッションが終了するまで、累積された状態で常に維持されます。
